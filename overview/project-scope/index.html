    The Path (AI-Pi) Project Scope¶

    Core Features¶

    1. Edge AI Gaming¶

    • Local AI processing on Raspberry Pi 5

    • AI HAT+ hardware acceleration

    • Adaptive difficulty system (0.2-0.95 range)

    • Basic performance monitoring

    2. Turn-Based Combat¶

    • Strategic class-based system

    • Basic buff/debuff mechanics

    • Turn-based actions

    • Status effects

    3. Battle System¶

    • Basic battle state tracking

    • Turn management

    • Win/loss recording

    • Simple stats tracking

    4. User Features¶

    • User authentication

    • Battle creation/joining

    • Basic battle history

    • Forfeit option

    Technical Objectives¶

    Edge AI Implementation¶

    Technical Requirements¶

    Hardware¶

    Platform:
  device: "Raspberry Pi 5"
  ram: "8GB"
  storage: "32GB+"
  accelerator: "AI HAT+"
  cooling: "Active"

    Software¶

    Stack:
  server: "Go 1.21+"
  client: "TypeScript/React"
  ai: "Python 3.11+"
  database: "MongoDB"

Dependencies:
  - WebSocket

  - TensorFlow Lite

  - React/Material-UI

  - Hardware drivers
